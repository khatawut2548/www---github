คอร์ดพื้นฐาน

    ​ วันนี้ Guitar chords มีคอร์ดเบื้องต้นอยู่ 6 คอร์ด จะมาบอกวิธีการจับคอร์ดกีตาร์อย่างถูกต้องแบบง่ายๆกันนะครับ

    1. คอร์ด G

    ( 1 ) = นิ้วชี้กดสายที่ 5 เฟร็ต 2
    ( 2 ) = นิ้วกลางกดสายที่ 6 เฟร็ต 3
    ( 3 ) = นิ้วชี้นางกดกดสายที่ 1 เฟร็ต 3
    ( 0 ) = สายที่ 2 เป็นสายเปล่า หรือ สายเปิด Open String
    ( 0 ) = สายที่ 3 เป็นสายเปล่า หรือ สายเปิด Open String
